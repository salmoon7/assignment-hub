Objective

Features to Include

Homepage:

  • Introduction to the Edutech app.
  • Navigation menu for easy access to different sections.
  • Brief overview of the app's functionality.

HTML Tags to Use

  • <header> for the header section.
  • <nav> for the navigation menu.
  • <section> for the main content area.
  • <footer> for the footer section.

CSS Styling

  • Style the header, navigation menu, and footer to make them visually appealing.
  • Use CSS to create a responsive layout that adapts to different screen sizes.

Courses Section

  • Display a list of available courses with brief descriptions.
  • Include thumbnails or icons for each course.

HTML Tags to Use

  • <ul> and <li> for the list of courses.
  • <img> for course thumbnails or icons.

CSS Styling

  • Style the course list to make it visually appealing.
  • Use CSS to align thumbnails and descriptions properly.

Course Details Page

Provide detailed information about each course.

  • Include course title, description, instructor name, and enrollment button.

HTML Tags to Use

  • <h2> for the course title.
  • <p> for the course description and instructor name.
  • <button> for the enrollment button.

CSS Styling

  • Style the course details page to enhance readability.
  • Use CSS to make the enrollment button stand out.

About Us Page

Information about the Edutech app creators or organization.

  • Brief description of the app's mission and goals.

HTML Tags to Use

  • <h2> for the page title.
  • <p> for the content.

CSS Styling

  • Style the About Us page to match the app's overall design.
  • Use CSS to format text and add visual elements.

Additional Instructions

  • Ensure that your HTML code is semantic and follows best practices.
  • Use external CSS stylesheets to keep your code organized and maintainable.
  • Test your app on different devices and browsers to ensure compatibility.
  • Consider user experience and accessibility principles while designing your app.

Submission Guidelines

  • Submit your HTML and CSS files in a zip folder.
  • Include a brief explanation of your design choices and any challenges you encountered.
